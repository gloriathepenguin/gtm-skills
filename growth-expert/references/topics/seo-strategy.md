# SEO 策略

## 核心框架

### 宗源（YouMind 增长工程师）

**核心指标：Domain Rating (DR)**
- DR 是 Ahrefs 的域名权威度评分（0–100）
- **DR 50+ 之前，不要去竞争通用信息词**，先蹭热门模型词（Sora/Seedance/Nano Banana）建立 DR
- DR 分数越高，博文越容易在非品牌信息词搜索中排名靠前

**哥飞原则**（宗源引用）：花 40% 时间挖需求，20% 时间开发，40% 时间宣传推广

## 外链建设

| 来源 | 方式 | DR 效果 | 成本 |
|------|------|---------|------|
| theresanaiforthat.com | 提交工具 | 一个外链 | 几十刀 |
| submitdirs.com | 批量目录提交（3个月） | +10 DR | 付费服务 |
| 集合站 | 高价值资源引发自然引用 | 数千外链 | 内容成本 |
| GitHub 仓库（200+ Stars） | 仓库 README 外链 | DR 96 Do-follow | Stars 积累 |

**GitHub 外链是性价比最高的策略**（宗源）：
- GitHub 本身 DR 96–97
- 超过 200 Stars → GitHub 在仓库页面生成 Do-follow 外链
- YouMind-OpenLab 旗下 13 个仓库、总 11.8K Stars，带来多个 DR 96+ 外链

## 关键词策略

**早期（DR 低时）**：
- 找新词、长尾词、一搜还没工具页承载的词
- 加 `?gl=xx` 参数切地区，很多工具站不做多语言优化
- 搜到工具页但域名 DR 不高的，可以尝试竞争

**中期（DR 稳定后）**：
- 蹭热门模型词写博文（如 Nano Banana、Seedance）
- 参考 skywork.ai：蹭模型词博客贡献整站 30% 流量

**避坑**：
- ❌ DR 低时挑战大词（如 youtube transcript），域名包含关键词的竞品整站吊打你单页
- ❌ 为了 SEO 工具页单独做匿名使用逻辑（研发投入大，注册转化率做好就够）

## 注册转化

**粗暴做法**：页面顶部 + 底部多放 CTA（天工 AI 的做法）

**精细做法**：
- One-Tap Login（右上角自动弹 Google 授权），可提升 3–8% 注册转化
- 福格行为模型：行为 = 动机 + 提示 + 能力——找到动机强的用户，降低能力门槛，增加提示
- 案例：加红点提示让点击率出现大转折

**受众匹配是前提**：
- YouMind 提示词站注册转化率 20%（来的是 AI 创作者 = 目标用户）
- Wordware Twitter Personality 注册转化率 3–4%（来的是普通用户，不是 AI 开发者）
